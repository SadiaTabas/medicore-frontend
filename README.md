# MediCore Hospital Management System (Frontend)

A modern web-based frontend for the MediCore Hospital Management System built using **Next.js** and **Tailwind CSS**. This application provides a responsive and user-friendly interface for doctors to manage profiles, appointments, and authentication features efficiently.

---

## Features

### Authentication

* Doctor Registration Page
* Secure Login & Logout
* Session-based Access Control

---

### Doctor Dashboard

* View appointment list
* Track upcoming schedules
* Manage consultation overview

---

### Profile Management

* Create and update doctor profile
* Add hospital and address information
* Upload supporting documents

---

### Appointment Management

* View patient appointments
* Check appointment details
* Monitor consultation schedules

---

### Doctor Directory

* Browse registered doctors
* View specialization details
* Check experience levels

---

## UI/UX Features

* Fully responsive design (mobile + desktop)
* Clean medical dashboard interface
* Fast navigation using Next.js routing
* Styled with Tailwind CSS

---

## Tech Stack

* Next.js (Frontend Framework)
* React.js
* Tailwind CSS
* TypeScript
* Axios (API Communication)

---

## Installation Guide

### Clone Repository

```bash id="fe1"
git clone https://github.com/your-username/medicore-frontend.git
```

### Install Dependencies

```bash id="fe2"
npm install
```

### Run Development Server

```bash id="fe3"
npm run dev
```

Frontend will run at:

```text id="fe4"
http://localhost:3000
```

---

## Environment Variables

Create a `.env.local` file:

```env id="fe5"
NEXT_PUBLIC_API_URL=http://localhost:5000
```
